# WaspMOT

**WaspMOT: A Benchmark for Long-Term Multi-Object Tracking of *Trichogramma* Wasps**

*Accepted at AVSS 2026*

This repository will provide instructions for accessing, setting up, and using the WaspMOT dataset accompanying the AVSS 2026 paper.

The repository is currently under preparation. The dataset will be released as the conference date approaches.
